# 💳 Bankist App (Vanilla JS)

A simple banking app built with **JavaScript, HTML, and CSS**.
Simulates basic banking operations with a clean UI.

🔗 Live Demo

👉 https://linschmitz.github.io/Bankist/

## 🚀 Features

- 🔐 Login with username & PIN
- 💸 Transfer money between accounts
- 🏦 Request loans
- ❌ Close account
- 📊 View balance, transactions, and summary
- 🔃 Sort transactions

## 🧠 Concepts Used

- DOM manipulation
- Array methods (`map`, `filter`, `reduce`, `find`, `sort`)
- Event handling
- Data transformation
